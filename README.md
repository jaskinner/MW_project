# Text Redactor App

## Overview
A simple Express.js app for redacting words from text files.

## Installation
```bash
git clone https://github.com/jaskinner/MW_project.git
cd project
npm install
```

## Running
Start the server:
```bash
npm start
```
Visit `http://localhost:3000` to use the app.

## Features
- Upload text files.
- Input words for redaction.
- Download redacted files.

## Testing
Run tests with:
```bash
npm test
```
